# ðŸ“Š What is a Data Mart?

A **Data Mart** is a smaller, focused version of a Data Warehouse.  
It contains data for one specific department or business area.

---

## ðŸ”¹ Simple Definition

A **Data Mart** is a structured subset of data designed for a specific team, such as:

- Sales  
- Finance  
- Marketing  
- HR  

---

## ðŸ”¹ Example

Imagine a company has:

- Sales data  
- HR data  
- Finance data  
- Inventory data  

All of this is stored in a **Data Warehouse**.

Now:

- The **Sales team** only needs sales metrics.  
- The **HR team** only needs employee data.  

So we create:

- ðŸ“Š **Sales Data Mart**  
- ðŸ’° **Finance Data Mart**  
- ðŸ‘¥ **HR Data Mart**

Each team gets only the data they need.

---

## ðŸ”¹ Data Warehouse vs Data Mart

| Data Warehouse | Data Mart |
|----------------|------------|
| Enterprise-wide | Department-specific |
| Large & complex | Smaller & focused |
| Stores all company data | Stores filtered data |
| Used by many teams | Used by one team |
